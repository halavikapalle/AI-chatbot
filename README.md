🤖 Rule-Based AI Chatbot
📌 Overview

This is a simple rule-based chatbot built using basic programming concepts. It responds to predefined user inputs using conditional logic and runs in a continuous loop until the user exits.

The project demonstrates foundational concepts of Artificial Intelligence, control flow, and decision-making logic.

🚀 Features
Responds to basic greetings like hi, hello, hey
Handles exit commands like bye, exit, quit
Continuous conversation loop until user terminates
Simple rule-based logic using if-else conditions
Lightweight and beginner-friendly AI simulation
🧠 How It Works

The chatbot uses:

Predefined keyword matching
if-else decision-making structure
A loop to continuously take user input
Exit condition to stop execution
